# Sentiment Analyzer 🚀

An AI-powered Sentiment Analysis web application built using **React** and **Vite**.  
This project analyzes text sentiment and visualizes results through interactive charts and a modern responsive interface.

---

## ✨ Features

- 🔍 Text Sentiment Analysis
- 📊 Interactive Charts & Visualizations
- ☁️ Word Cloud Visualization
- 📁 CSV File Upload Support
- ⚡ Fast & Responsive UI
- 🎨 Modern User Interface

---

## 🛠️ Tech Stack

- React
- Vite
- JavaScript
- Chart.js
- Three.js
- HTML5
- CSS3
